This prompt enables you to design navigation menus using ASCII art characters and pixel-art styling, creating authentic terminal-style interfaces reminiscent of early computing BBS systems, DOS applications, and text-based user interfaces. It supports various menu types including horizontal menu bars, vertical sidebars, dropdown menus, breadcrumb trails, and tab navigation, all rendered with classic box-drawing characters, decorative ASCII elements, and monospaced alignment. The prompt provides detailed specifications for visual aesthetics, character sets, color schemes, state management (normal, hover, active), and responsive behavior, making it perfect for creating nostalgic retro-themed web applications, vintage terminal-inspired interfaces, or authentic text-based navigation systems that evoke the charm of early computing era.
